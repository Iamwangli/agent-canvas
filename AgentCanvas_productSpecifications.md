# AgentCanvas 最终产品规格（实现版）

**文档更新日期**：2026-05-11  
**版本**：2.4（新增节点摘要、折叠、对齐、侧边栏收起、节点背景优化）

---

### 一、产品定位

无限画布，用户通过树形结构的对话节点与 Agent 根节点，构建多智能体协作的推理链。每个节点代表一轮完整的单次问答（用户→AI 或 AI→AI）。上下文沿父链向上游追溯，隐藏节点可选择性排除自身内容，但不影响子节点继续向上索引。**支持多会话管理**（文件模型），所有画布数据自动持久化到浏览器本地存储。

---

### 二、节点类型与视觉规范

| 节点类型 | 形状 | 样式 | 内容结构 | 其他 | 状态 |
|---------|------|------|----------|------|------|
| **Agent 根节点** | 椭圆 | 浅蓝色填充，黑色边框，文字加粗 | 显示 Agent 名称（可编辑）、模型标识 | 不可删除 | ✅ |
| **对话节点（普通）** | 圆角矩形 | `bg-gray-100` 填充，黑色加粗边框（`border-2 border-black`） | 上半区问题，下半区回复，可显示附件标签 | 宽度可拖拽调整（200-1000px），高度自适应 | ✅ |
| **对话节点（AI 自动创建）** | 同上 | 金色虚线边框 + 淡黄色背景，呼吸动画 | 同上 | 高亮提醒 | ✅ |
| **隐藏节点** | 同上 | 灰色虚线边框，半透明填充 | 同上 | 内容可见，但不参与上下文 | ✅ |

- **连线**：有向贝塞尔曲线（子→父），四个方向 Handle，末端箭头。默认深灰色（`#374151`），线宽 2.5px。✅
- **流光线动画**：发送消息等待回复期间，从当前节点沿父链到根节点的连线呈现金色虚线流动效果，节点边框同步脉冲。✅
- **尺寸**：宽度可拖拽调整（200-1000px），高度随内容自适应，支持 Markdown。✅

### 三、核心交互操作

#### 3.1 初始化画布与多会话管理 ✅

- **会话（文件）模型**：每个会话是一个独立的画布状态。所有会话保存在 `files` 数组中，当前活动会话由 `currentFileId` 标识。✅
- **自动持久化**：所有会话数据自动保存到 `localStorage`（键名 `agent-canvas-files`），通过 `hasHydrated` 确保恢复完毕，刷新后不丢失。✅
- **初始化**：首次打开自动创建“未命名会话”，并加载后端 Agent 列表生成根节点。✅
- **历史侧边栏**：左侧显示所有历史会话列表，显示名称和创建时间，支持**恢复**和**删除**。删除会话时至少保留一个。✅
- **侧边栏收起**：标题栏右侧按钮可收起/展开侧边栏，收起时保留标题行，列表隐藏，节省画布空间。✅
- **会话重命名**：双击历史侧边栏中的会话名即可内联编辑，Enter 保存，Escape 取消。✅

#### 3.2 创建对话节点 ✅

- **手动创建（双击节点）**：双击任意节点（Agent 或对话节点），在**下方**创建空白子节点，自动连线（子→父，带箭头）。新节点进入输入模式，输入框聚焦。✅
- **右键创建子节点（方向）**：右键菜单二级菜单选择上方/右方/下方/左方，子节点出现在指定方向并自动使用对应 Handle 连接。只有已回复的对话节点或 Agent 根节点允许创建子节点。✅
- **输入与发送**：输入问题，回车或点击发送。系统收集上下文（沿父链向上，跳过隐藏节点），调用对应 Agent API，回复显示在下半区。发送期间触发流光线动画。发送后节点变为只读。✅
- **文件上传**：支持常见文本格式（.txt/.md/.json/.py/.html/.css 等）。上传后显示文件标签，不填充输入框。发送时内容置于问题之前（利用前缀缓存），节点保存的 `question` 仅含用户文本。附件列表持久化显示于问题下方。✅

#### 3.3 AI 自动创建节点 ✅

- 工具栏“🤖 AI自动节点”开关，默认开启，持久化。✅
- 当回复中被检测到需要其他 Agent 协助时，自动创建高亮节点并自动发送。✅
- 防滥用：同一父节点下连续自动创建超过 3 次时弹窗确认。✅

#### 3.4 节点操作（手动）

| 操作 | 方法 | 行为 | 状态 |
|------|------|------|------|
| **调整宽度** | 拖拽节点边缘 | 宽度 200-1000px，宽度值持久化 | ✅ |
| **移动节点** | 拖拽主体 | 连线自动重绘 | ✅ |
| **重新指定父节点** | 从子节点 Handle 拖线至新父节点 | 断开原父节点，连接新父节点，禁止循环；只允许子→父方向 | ✅ |
| **删除节点** | 右键菜单 → 删除 | 弹出确认框，删除后子节点自动连接至原父节点（若无父节点则变为孤立） | ✅ |
| **隐藏/取消隐藏** | 右键菜单 | 菜单根据状态互斥显示，隐藏节点不参与上下文，但不阻断子节点追溯 | ✅ |
| **复制节点** | 右键菜单 → 复制节点 | 复制问题和回复、摘要、附件、宽度，生成自由节点 | ✅ |
| **复制内容** | 右键菜单 → 复制内容 | 将问题 + 回复写入系统剪贴板 | ✅ |
| **折叠/展开** | 右键菜单 → 折叠节点/展开节点 | 折叠后问题和回复仅显示前 3 行（渐变遮挡），底部出现“显示全部”按钮 | ✅ |
| **对齐到父节点** | 右键菜单 → 对齐到父节点 | 根据连线方向调整子节点的水平或垂直位置，使连线为直线 | ✅ |
| **查看摘要** | 右键菜单 → 查看摘要 | 弹窗显示节点的问题与回复摘要 | ✅ |

#### 3.5 连线操作 ✅

- 手动连线只允许子→父，反向被拒绝并提示。双击删除连线，子节点变为孤立节点。
- Handle 信息（`sourceHandle`/`targetHandle`）持久化，刷新后连线位置不变。
- 连线箭头、颜色、线宽均可通过 CSS 调整。

#### 3.6 缩放 ✅

- 支持 `minZoom={0.05}`，极缩视角下可查看整体结构。已移除 LOD 简化渲染，始终显示真实内容。

### 四、上下文索引规则 ✅

- **路径**：从节点出发沿父链向上收集祖先节点的问题和回复。
- **隐藏节点**：跳过隐藏节点的内容，继续向上。
- **摘要增强**：若祖先节点含有 `summary` 字段，上下文传递给大模型时，该节点的上下文顺序为 **摘要（system）→ 问题（user）→ 回复（assistant）**。摘要对画布不可见，但右键可查看。
- 孤立节点无法发送，提示先连接到 Agent 根节点。

---

### 五、Agent 路由与 API 调用 ✅

- 每个对话节点沿父链查找最近的 Agent 根节点，调用对应大模型 API。
- 支持 DeepSeek、Qwen、OpenAI 等（需配置 Agents）。
- 摘要生成请求使用 `skipAutoAction=true`，避免触发自动节点。

### 九、技术实现要点 ✅

- 持久化：Zustand + `persist` + `onRehydrateStorage`，`localStorage`。
- 上下文收集：`collectContext` 增加 `summary` 返回。
- 连线持久化：节点数据中保存 `sourceHandle`/`targetHandle`。
- 摘要生成：回复成功后后台调用大模型生成摘要，保存至 `node.summary`。
- 折叠/对齐：均为节点数据操作，对齐根据 Handle 方向计算位置。

### 十、边界情况与约束

| 场景 | 处理方式 | 状态 |
|------|----------|------|
| 复制节点无摘要 | 已修复，复制时携带 `summary` | ✅ |
| 折叠节点内容溢出 | 使用 `max-height` + 渐变遮罩 | ✅ |
| 对齐后节点跳跃 | 仅按连线方向调整对应轴 | ✅ |
| 摘要生成失败 | 静默失败，不影响主流程 | ✅ |
| 其他 | 略（之前已覆盖） | ✅ |
